## loca_text(string)

|Arg|Type|Description|
|:--|---|:--|
|locatext|String|The text to find in the localization.|

### Returns: String
### Example:
```gml
text = loca_text("before_fixing_heart_prompt_sub");
show_message('Here: ' + text);
```

This function accepts a string to find in the localization.
